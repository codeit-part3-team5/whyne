# 🍷 WHYNE

WHYNE는 사용자들이 다양한 와인에 대한 리뷰를 보고, 검색·필터링하며, 직접 리뷰를 남기고 와인을 등록할 수 있는 리뷰 기반 와인 플랫폼입니다.

- 종료일: 2025.06.24
